一个文科生的工程师之路

发表于 2017-07-31 | 更新于 2017-07-31 | 分类于 行业 | |

什么样的人适合成为“程序员”？先来看两张图。

此图一出，立即有人调侃：这位同学，看来你很适合做“程序员”啊，毕业记得来华为报到！

实际上，程序员真的是这样吗？对此，博主要说NO，毕业多年，接触的程序员上百人，这样的人有没有，有，但极少。为什么“程序员”会被黑？因为他们沉默、低调、闷骚，不擅长辩解，薪资水平略高，容易成为被黑对象。而从业人员基本出自理工科，该科男多女少，比例失调，“基佬“众多，解闷打趣，久而久之，走入社会，接触女性渐多，容易出现一些失礼情况；这就说到了情商，没人是天生的高手，都是后期通过不断学习和修正达到的，难道其他行业没有这样的人吗？

工程师被戏称为”程序员“，且分”男猿“和”女猿“（人类的祖先也是猿，是不是有点造物主的意思），一般认为来自理工科，其实非科班出身的、生物、化学专业的也不少，博主虽然是科班出身，但实际上却是一名文科生，而非通常认为的理科生，至于怎么走上这条”不归路“，话休烦絮，且听接下来分解。

初中开始接触政、史、地和理、化、生：初一政治、历史、地理，初二生物、物理，初三化学，懵懂少年的求学时代，不知道什么叫喜好，只听得老师讲”有志者，事竞成，破釜沉舟，百二秦关终属楚；苦心人，天不负，卧薪尝胆，三千越甲可吞吴“、”勤能补拙“，死学硬背，外加理解，政治、历史、地理成绩都还不错，而生物、化学略有兴趣，物理一上课就被”催眠“，效果不要太好！尤其初二时觉得物理这么难学，以后如果选科可以不读物理最好了，事实证明，如果这个想法一直坚持就对了。由于其他科成绩不错，物理分数占比很小，也以一分之差中考志愿二中（市排名第三，不算差），结果调剂成功，似乎报了点侥幸的心理。高考会不会也这样呢？

高一开始，文综、理综一起考，总分850能考720，班主任是数学老师，所以引导大家都去报理科，一个口号”能读理科，不读文科，文科出来找不到工作“。为什么要说引导，班主任的奖金跟考上本科和重点的学生数量来算，除了老师之外，其他一些认同观念的同学也推波助澜，结果80%以上同学都读了理科；然而一到高二，物（物理、生物）效马上出现，原来可以轻松前10名，现在要很努力才能保持15名左右，后来实在听不懂、弄不会，时间一长，就真的放弃了，专注语、数、外和化学，高三全部复习一遍，结果高考离志愿差11分，这个时候才真的意识到，如果自己没兴趣，真的不要勉强！至于什么叫兴趣，怎么看出来，接下来，我们可以举个例子：

历史三道考题，”黄花岗起义的第一枪是谁开的，第二枪是谁开的，第三枪是谁开的，选项是A宋教仁B孙中山C黄兴D徐锡麟“，如果你是个文科生，读过这段历史，记得黄兴连开三枪，揭开黄花岗起义序幕，那么这题必须是送分题，三题分数全到手，信心十足！相反，如果你是个理科生，对历史不感兴趣，那么你会想到底是谁开的呢，都像啊，至少也记得一个徐锡麟连开三枪刺杀恩铭，要不一个A，一个B，一个C，至少能对一个呀！对的，不感兴趣你就是这样答题的，博主答物理题就类型让理科生答文科题，你根据就没办法开启大脑存储功能！感兴趣的有没有必要你都会记下来，遇到问题会举一反三，不感兴趣打死都想不起来，兴趣就是这么神奇，别不信！至于老师前面说的格言对不对，对肯定的对的，老师说的能错吗？关键是说的对就一定要听吗，听了就要一定去做吗？这个很考验自我决策能力，对于一直依赖家庭成长的小孩来说，做决策都是作文写记叙文还是议论文，课余时间复习数学还是英语，关于兴趣这个认知问题，小孩做这样的决定相对来说，是稚嫩的，不成熟的。

同样，物理考题也是如此，电路问题，一根电线通过多少电压，电线会变成电阻、电感、电容？从小对电路不感冒的博主，你说常识性的东西，重力、向心力、电磁力、行星引力等方面了解，简单的题会做，但题目一变种，就像上面的历史题一样，懵了是常有的事，这也就是上面讲初二时的一次感悟：能不学自己不感兴趣的就千万不要学呀，影响重要关口；事实就是这样，第1年高考是这么被影响的，马上复读进入培优班（本来要进两三百人的复读班，结果被接下来的班主任忽悠，说复读班已满-由此也可以看出，老师们的急功近利心；感言：市排名第1的中学确实好，关注学生的方向，第2压力大，第3可以说有些扭曲《学校有复读4年，每年高考600多分的同学》），可以说将物理、生物上在高二落下的知识全部学懂，然而考试仍然考不好，这就是兴趣的神奇所在！

也许有人会问，博主你就那么容易被影响吗？这样你以后的人生道路上，能做出成就吗？今天就好好回答一下你这个问题，选择理科，当时确实还有个小私心：读了理科之后，文科方面的东西想学很容易，但读了文科之后，理科方面东西没有压力就不会去学了；然而并没有过分关注后果，导致高中4年，3年兴趣压抑，加上家里做米面生意-静电引起的灰尘较多，学习环境不好，压力山大，精神衰弱，几乎崩溃，能想到的勤奋和苦学都用上，然而效果接近于0，本着专业优先、学校第二的原则，结果第2年离志愿又以1分之差。而高考这件事，严重教会博主一件事：顺势而为。

中间小插曲，高考结果，在家无事，天天去附近操场打篮球，家里本着”艺多不压身“的理念，给我报了一个计算机班，学习五笔、办公三剑客、上网。说起来汗颜，中学会用电脑玩游戏，做flash，高中申请QQ，直到毕业，才真正学会电脑都做些有意思的事情。

接着第3年踏上复读之路，去的是市排名第1的中学，校园环境跟大学无异，老师开放，下课教室空无一人，全去操场（与市第3名的下课后，教室仍旧人头攒动大相径庭），于是下决心，要努力争取读文科，结果一而二，再而三的被班主任拒绝，于是两周时间，晚自习睡觉不想学，被约谈家长，看他实在不想再读，要不就去读个本市理工大学的计算机吧，男孩子动手能力强，父亲征求我的意见后，考虑到英文和计算机这个两个优势，于是再次补录，以绝对优势进入软件学院的计算机专业。

这就是我的”程序员“之路，把选择的心路历程公布出来，希望大家能多关注中学生的心理和选择问题，帮助他们做出属于自己的正确选择，减少”背篙撵船“的事情发生，最后祝每位读者都成为栋梁之才！

FAQ

  • 问题1：什么样的人可以当程序员？
    对计算机感兴趣，或者科班出身，计算机学习的是编程语言，属于高级语言，课程设置如右：计算机科学与技术，只要操作能力强、逻辑思维可以，不反感数学均可。

  • 问题2：程序员就业好吗？
    据专家介绍，人才缺口几百万，大家眼里的高薪就是最好证明。
    我国数据人才缺口约150万-2016.5.31
    前程无忧显示全国IT人才缺口大 加薪幅度领先-2014.12.2
    报告显示互联网人才缺口大 近半为硕士-2015.8.7

  • 问题3：本科可以，还是需要硕士？
    君不见，BAT年年提高面试标准，一般考个硕士更好，痴迷计算机者除外，211/985尤佳，其次本科（注意不分一、二、三本），重要！重要！重要！本科即可，不要求几本（（高的毕业pack20万）），除非985/211外，接着硕士（高的毕业pack30万）、博士（高的毕业pack50万），以上数字均作参考；其次学校重要于专业，你可以读双学位或者自学均可，求职不怎么看重专业。

  • 问题4：怎样切入职场？
    学好大学课程，主要是基本知识和大局观，注意网申和实习机会，提前准备面试；另外算法、安全类大赛也帮助很大，排名靠前直接录用的也不在少数；如果不幸毕业才找工作，往往毕业即失业，被当作社招，竞争力显弱，只能靠运气。
