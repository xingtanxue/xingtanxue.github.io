XingtanXue网站由来

发表于 2017-01-26 | 更新于 2017-01-31 | 分类于 感悟 | |

有一天，

群里有人用Hexo做一个博客，并分享出来。

而自己之前买了N多域名，用WordPress做过一个功能一般的网站

所以打算再做一个，于是网上搜了一下。

使用github做服务器，花两天时间，

用github、python、nodejs等工具完成。

工具：
Lean Cloud:统计UV
多数：做评论
百度：做数据统计

安装过程：
npm install hexo-cli -g -d
hexo init
npm install

注意事项：
hexo是一种新的语法，命令后的冒号必须有空格
hexo new page “标签” 新建一个栏目
hexo new “我有一个中国梦” 新建一篇文章
用标准格式可以显示“标题”、“标签”和分类：

文章格式：

title: 标题
tags: 标签

categories: 分类

网站部署：
hexo clean 删除本地的public文件
hexo g -d 生成并布署
hexo s 启动服务器本地查看 浏览器输入localhost:4000即可
npm install hexo-generator-feed –save #安装组件，自动生成atom.xml文件，方便别人订阅

2017.1.30
今天又把网站美化一下，使用Next主题，并成功加入多说评论，好朋友一起快乐的玩耍！
